#Weather App

A responsive web application built entirely with Tailwind CSS that provides current weather information for any city using the OpenWeatherMap API.

##Features

. Search for weather by city name

. Displays temperature, humidity, wind speed, and weather description

. Fully responsive design using Tailwind CSS

. Styling adapts dynamically to different weather conditions

##Technologies Used

. HTML5

. Tailwind CSS

. JavaScript

. OpenWeatherMap API
